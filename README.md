# Analisis Sintactico - Taller LPT

## Descripcion general

Este repositorio contiene la implementacion de tres ejercicios relacionados con analisis sintactico, enfocados en el estudio de gramaticas, algoritmos de parsing y comportamiento de expresiones aritmeticas.

---

## Estructura del repositorio

El proyecto se organiza en tres partes principales:

- Ejercicio 1:
  Implementacion de un analizador sintactico basado en descenso recursivo que genera arboles sintacticos para expresiones aritmeticas.

- Ejercicio 2:
  Comparacion entre el algoritmo CYK con complejidad O(n^3) y un parser lineal con complejidad O(n), evaluando numero de operaciones y tiempo de ejecucion.

- Ejercicio 3:
  Analisis de asociatividad y precedencia mediante modificaciones a la gramatica, mostrando como cambia el arbol sintactico para una misma cadena.

---

## Conceptos trabajados

- Gramaticas libres de contexto
- Analisis sintactico descendente
- Algoritmo CYK
- Complejidad temporal O(n^3) y O(n)
- Asociatividad de operadores
- Precedencia de operadores
- Arboles sintacticos
